
本网站基于 **Astro** 框架。

::github{repo="xinvxueyuan/xinvxueyuan.github.io"}

## 🌟 主题特性

### 🎨 设计与用户体验
- **现代优雅设计** - 简洁、极简的界面与美观的排版
- **完全响应式** - 针对移动端到桌面端的全设备优化
- **深色/浅色模式** - 自动主题切换，过渡流畅
- **精美排版** - 采用 JetBrains Mono 字体，提升可读性
- **流畅动画** - 页面切换与交互元素动画顺滑

### 🔍 内容与搜索
- **高级搜索** - 由 [Pagefind](https://pagefind.app/) 提供支持，快速且精准
- **增强型 Markdown** - 扩展语法，支持代码高亮与数学公式
- **交互式目录** - 长文自动滚动导航
- **RSS 订阅生成** - 自动生成订阅源，随时获取更新
- **阅读时长估算** - 预估文章阅读时间
- **文章分类** - 通过标签和分类组织内容



### 📱 特殊页面
- **番剧追踪页** - 记录你的追番进度与评分
- **友链页面** - 用精美卡片展示友站
- **日记/动态页** - 像社交媒体一样分享生活点滴
- **归档页** - 以时间轴方式整理所有文章
- **关于页** - 可自定义的个人介绍（本页！）

### 🛠 技术特性
- **增强代码块** - 由 [Expressive Code](https://expressive-code.com/) 驱动
- **数学公式支持** - 使用 KaTeX 渲染 LaTeX 数学表达式
- **图片优化** - PhotoSwipe 图库与懒加载
- **SEO 优化** - 内置站点地图与元标签，提升搜索可见性
- **性能优化** - 缓存与优化，加载更快
- **评论系统支持** - 可集成 Twikoo 评论

### 🎯 高级 Markdown 特性
- **提示与警告框** - 使用 `> [!NOTE]`、`> [!TIP]`、`> [!WARNING]` 显示美观的信息框
- **数学公式** - 通过 `$行内$` 和 `$$块级$$` 语法书写 LaTeX 数学
- **GitHub 卡片** - 用 `::github{repo="user/repo"}` 嵌入仓库卡片
- **语法高亮** - 代码高亮并显示行号
- **一键复制代码** - 便捷的代码复制按钮


*由 Astro 构建，灵感源自现代网页设计理念，怀着热爱开发。*
